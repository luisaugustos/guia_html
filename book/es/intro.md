# Introducción a HTML

Bienvenido a la **Guía HTML para Ingeniería Informática**.

Este libro te enseñará lo esencial que necesitas saber como estudiante de Ingeniería Informática sobre **HTML (HyperText Markup Language)**. No es un manual exhaustivo, sino una introducción pragmática a los conceptos y herramientas que todo ingeniero informático debe dominar.

## ¿Qué es HTML?

HTML es el **lenguaje de marcado estándar** para crear páginas web. Proporciona la estructura y el significado del contenido web, mientras que CSS controla la presentación y JavaScript añade interactividad.

Como ingeniero informático, entender HTML es fundamental porque:

- **Es la base de la web**: toda página web, aplicación web y contenido digital moderno se construye sobre HTML.
- **Es simple pero poderoso**: puedes crear contenido estructurado de forma clara y semántica.
- **Te permite comunicarte con otros profesionales**: diseñadores, desarrolladores frontend y especialistas en accesibilidad hablan el lenguaje de HTML.
- **Abre puertas**: comprender HTML es el primer paso hacia desarrollo web, aplicaciones progresivas (PWA), documentos electrónicos y más.

## ¿Qué aprenderás?

En este libro cubriremos:

1. **Conceptos Fundamentales** — Qué son las etiquetas, atributos, y cómo se estructura un documento HTML.
2. **El Documento HTML** — La anatomía de una página: DOCTYPE, head, body, meta tags, favicon y SEO.
3. **Elementos Semánticos** — Cómo usar las etiquetas correctas para dar significado a tu contenido (encabezados, párrafos, listas, citas).
4. **Enlaces e Hipervínculos** — Cómo conectar documentos y recursos con URLs, rutas absolutas y relativas.
5. **Tablas** — Cómo estructurar datos tabulares correctamente.
6. **Tu Primer HTML** — Un proyecto práctico para poner todo junto.

## Cómo usar este libro

- Cada capítulo incluye **ejemplos prácticos** que puedes copiar y experimentar.
- Los conceptos están ordenados **de lo simple a lo complejo**.
- Se recomienda seguir el orden secuencial: cada capítulo se construye sobre los anteriores.

## Herramientas que necesitarás

- Un **editor de código** (VS Code, Sublime Text, Atom, etc.)
- Un **navegador web** (Chrome, Firefox, Safari, Edge)
- **Línea de comandos** básica (Git Bash, PowerShell o Terminal)

Eso es todo. HTML no requiere instalaciones complicadas.
